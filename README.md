# Makakatrade

Modern trading platform built with React and TypeScript.

## Quick Start

```sh
cd frontend-project
npm install
npm run dev
```

## Tech Stack

- React 19 + TypeScript
- Redux Toolkit
- Vite
- Tailwind CSS
- React Router

## Commands

```sh
npm run dev      # Development server
npm run build    # Production build
npm run lint     # Code linting
```

## Structure

```
frontend-project/
├── src/         # Source code
├── public/      # Static assets
└── dist/        # Build output
```
